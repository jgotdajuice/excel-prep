**Step 3 — Implement `src/engine/grader.ts`:**

```typescript
import { DetailedCellError } from 'hyperformula';
import type { AnswerCell, GradeResult } from '../types';

export function gradeCell(
  cellValue: unknown,
  answerCell: AnswerCell,
): GradeResult {
  if (cellValue instanceof DetailedCellError) {
    return { status: 'error', errorCode: cellValue.value };
  }

  const { expectedValue, tolerance = 0.01 } = answerCell;

  if (typeof expectedValue === 'number' && typeof cellValue === 'number') {
    return Math.abs(cellValue - expectedValue) <= tolerance
      ? { status: 'correct' }
      : { status: 'incorrect', gotValue: cellValue, expectedValue };
  }

  return cellValue === expectedValue
    ? { status: 'correct' }
    : { status: 'incorrect', gotValue: cellValue as string | boolean, expectedValue };
}
```

**Step 1 — Create `src/store/challengeStore.ts`:**

Zustand store managing the full challenge session state. Use `create` from `zustand` (already installed, Zustand 5.x).

```typescript
import { create } from 'zustand';
import type { Challenge, ChallengeStatus, GradeResult } from '../types';
import { gradeCell } from '../engine/grader';

interface CellGrade {
  row: number;
  col: number;
  result: GradeResult;
}

interface ChallengeStore {
  // Data
  challenges: Challenge[];
  currentIndex: number;
  statuses: ChallengeStatus[];
  cellGrades: CellGrade[];

  // UI state
  hintVisible: boolean;
  explanationVisible: boolean;
  isLocked: boolean;            // Grid locked after grading
  startTime: number | null;     // Timer start timestamp
  elapsedSeconds: number;

  // Actions
  loadChallenges: (challenges: Challenge[]) => void;
  setChallenge: (index: number) => void;
  gradeCellAction: (row: number, col: number, cellValue: unknown) => void;
  showHint: () => void;
  toggleExplanation: () => void;
  retry: () => void;
  skip: () => void;
  nextChallenge: () => void;
  prevChallenge: () => void;
  tick: () => void;             // Called by timer interval
}
```

Implementation notes:
- `loadChallenges`: sets challenges array, resets statuses to all 'unattempted', resets currentIndex to 0
- `setChallenge(index)`: sets currentIndex, resets cellGrades/hintVisible/explanationVisible/isLocked for that challenge, starts timer
- `gradeCellAction(row, col, cellValue)`: calls `gradeCell(cellValue, answerCell)` where answerCell is found by matching row/col against `challenges[currentIndex].answerCells`. Pushes result to `cellGrades`. If ALL answer cells now have grades: determine overall status (all correct → 'correct', else 'incorrect'), update `statuses[currentIndex]`, set `isLocked = true`
- `retry()`: clears cellGrades, sets isLocked = false, resets status to 'unattempted', hides explanation
- `skip()`: sets status to 'skipped', advances to next challenge
- `nextChallenge()`: increments currentIndex (if not at end), calls setChallenge logic
- `prevChallenge()`: decrements currentIndex (if not at 0), calls setChallenge logic
- `tick()`: if startTime is set, update elapsedSeconds = Math.floor((Date.now() - startTime) / 1000)

Expose a `currentChallenge` derived getter: `challenges[currentIndex]` (or use it inline in components via the index).
