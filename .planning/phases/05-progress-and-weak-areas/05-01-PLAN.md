<task type="auto">
  <name>Task 2: Create progress selector functions</name>
  <files>src/store/progressSelectors.ts</files>
  <action>
Create `src/store/progressSelectors.ts` with these pure functions:

1. **`CategoryAccuracy` interface:**
   ```
   { category: string; tier: Tier; gridCorrect: number; gridTotal: number; drillCorrect: number; drillTotal: number; accuracy: number; totalAttempts: number }
   ```
   - `accuracy` = combined rate: `(gridCorrect + drillCorrect) / (gridTotal + drillTotal)` (simple ratio, no weighting — this is for display, not tier gating).
   - Only count 'correct' and 'incorrect' statuses for gridTotal (skip 'unattempted' and 'skipped' per Pitfall 5 from RESEARCH.md).
   - `totalAttempts` = gridTotal + drillTotal.

2. **`computeCategoryAccuracies(challenges: Challenge[], statuses: ChallengeStatus[], allAnswers: DrillAnswerRecord[]): CategoryAccuracy[]`**
   - Group challenges by `category` field.
   - For each category: count grid correct/total from statuses (only 'correct'/'incorrect' count), count drill correct/total from allAnswers filtered by challengeId belonging to that category.
   - Only return categories where totalAttempts > 0 (user has attempted at least one question).
   - Sort by tier order (beginner first, then intermediate, then advanced), then alphabetically within tier.

3. **`weakestCategory(accuracies: CategoryAccuracy[], minAttempts = 2): CategoryAccuracy | null`**
   - Filter to categories with totalAttempts >= minAttempts.
   - Return the one with lowest accuracy. Null if none qualify.

4. **`overallStats(challenges: Challenge[], statuses: ChallengeStatus[], allAnswers: DrillAnswerRecord[]): { totalChallengesCompleted: number; totalDrillAnswered: number; overallAccuracy: number; hintUsageCount: number }`**
   - totalChallengesCompleted = count of statuses that are 'correct' or 'incorrect' (not unattempted/skipped).
   - totalDrillAnswered = allAnswers.length.
   - overallAccuracy = (all correct across grid+drill) / (all attempted across grid+drill). Return 0 if no attempts.
   - hintUsageCount is passed through (caller reads from store).

Import Challenge, ChallengeStatus, Tier from `../types` and DrillAnswerRecord from `./drillStore`.
  </action>
