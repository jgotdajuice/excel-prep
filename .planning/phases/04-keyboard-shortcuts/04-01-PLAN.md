**Store interface (ShortcutStore):**
- **Config:** `osMode: OSMode`, `drillMode: DrillMode`, `drillDirection: DrillDirection`, `selectedCategory: ShortcutCategory | 'all'`
- **Session state:** `drillState: DrillState`, `queue: Shortcut[]`, `currentIndex: number`, `results: DrillResult[]`, `questionStartTime: number`, `timeRemaining: number` (seconds, for timed mode)
- **Feedback state:** `lastPressedKeys: string[]`, `lastCorrect: boolean | null`
- **Quit dialog:** `showQuitConfirm: boolean`

**Actions:**
- `setConfig(os, mode, direction, category)` — sets session config
- `startSession(shortcuts: Shortcut[])` — transitions idle→drilling, initializes queue/index/results/timer
- `submitAnswer(pressedKeys: string[])` — for Action→Keys direction: grades keypress answer, records result, transitions drilling→feedback. **Grading logic:** call internal `gradeKeys(pressed, shortcut, osMode)` — sorted set comparison (research Pattern 5). Ignore modifier-only presses (if pressedKeys is empty or contains only modifier names with no action key, do NOT grade).
- `submitMultipleChoiceAnswer(selectedShortcutId: string)` — for Keys→Action direction: compares selected shortcut id against current shortcut's id. Records result (correct if ids match), transitions drilling→feedback.
- `advanceToNext()` — feedback→drilling (next question) or feedback→summary (if last question). Resets `timeRemaining` to 7 for timed mode. Sets `questionStartTime = Date.now()`.
- `tick()` — decrements `timeRemaining` by 1 if `drillState === 'drilling' && drillMode === 'timed'`. If `timeRemaining` reaches 0, treat as timeout (same as incorrect — record result with `correct: false`, transition to feedback).
- `requestQuit()` — sets `showQuitConfirm = true`
- `confirmQuit()` — transitions to `summary` with partial results
- `cancelQuit()` — sets `showQuitConfirm = false`
- `resetToIdle()` — resets all session state to initial values, transitions to `idle`

**Pure grading function (internal, exported for testing):**
```typescript
export function gradeKeys(pressed: string[], shortcut: Shortcut, os: OSMode): boolean {
  const expected = os === 'windows' ? shortcut.keys.windows : shortcut.keys.mac;
  if (pressed.length !== expected.length) return false;
  const sortedPressed = [...pressed].sort();
  const sortedExpected = [...expected].sort();
  return sortedPressed.every((k, i) => k === sortedExpected[i]);
}
```

**Defaults:** `osMode: 'windows'`, `drillMode: 'practice'`, `drillDirection: 'action-to-keys'`, `selectedCategory: 'all'`, `drillState: 'idle'`, `timeRemaining: 7` (7 seconds per question in timed mode, per Claude's discretion within user's 5-10s range).

**CRITICAL:** Do NOT import from or modify `challengeStore.ts`. This is an entirely separate store.
