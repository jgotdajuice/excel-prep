  <name>Task 1: Extend types with Tier, drill fields, and DrillQuestion</name>
  <files>src/types/index.ts</files>
  <action>
Add to src/types/index.ts:

1. Export `Tier` type: `export type Tier = 'beginner' | 'intermediate' | 'advanced';`

2. Extend the `Challenge` interface with these NEW fields (keep all existing fields unchanged):
   - `tier: Tier` — authoritative learning path tier
   - `drillPrompt?: string` — short text prompt for drill mode with inline data (no grid references). Falls back to `prompt` if absent.
   - `drillAnswer: string` — exact expected answer string in drill mode
   - `drillAnswerScope: 'formula' | 'function'` — whether drill asks for full formula or just function name
   - `drillWrongOptions: string[]` — 3 wrong options for multiple choice mode

3. Add `DrillQuestion` interface:
```typescript
export interface DrillQuestion {
  challengeId: string;
  prompt: string;
  correctAnswer: string;
  answerScope: 'formula' | 'function';
  wrongOptions: string[];
  tier: Tier;
  category: string;
  explanation: string;
  correctFormula: string;
}
```

4. Add factory function:
```typescript
export function challengeToDrillQuestion(c: Challenge): DrillQuestion {
  return {
    challengeId: c.id,
    prompt: c.drillPrompt ?? c.prompt,
    correctAnswer: c.drillAnswer,
    answerScope: c.drillAnswerScope,
    wrongOptions: c.drillWrongOptions,
    tier: c.tier,
    category: c.category ?? 'General',
    explanation: c.explanation,
    correctFormula: c.correctFormula,
  };
}
```

5. Update the existing `difficulty` field type from inline union to: `difficulty?: Tier;`

Do NOT change any other existing types (CellAddress, SelectedCellState, CellContent, AnswerCell, ChallengeStatus, GradeResult).
